# ThunderHosts
[English](/README.md)/[中文](/README_cn.md)
### 咋用呢?
- `git clone https://github.com/guolifu/thosts.git`
- 修改/etc/hosts文件的权限
`sudo chown yourname:yourgroup /etc/hosts`
- `cd ./thosts`
- `php -S localhost:8000 `
- 在浏览器中输入 [http://localhost:8000](http://localhost:8000)
