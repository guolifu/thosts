[[English]](/README.md)/[[中文]](/README_cn.md)
### How to use it ?
- `git clone https://github.com/guolifu/thosts.git`
- `cd ./thosts && sudo php -S localhost:8000`
- Open the browser and enter URL [http://localhost:8000](http://localhost:8000)
